**Tablet (768-1023px):**
- 2-column grids (where desktop has 3-4)

- Touch-friendly but more compact than mobile

**Desktop (1024px+):**
- Full specifications as detailed above

- All interactive elements: Minimum 44×44px
- Adequate spacing between clickable elements (8px minimum)
- No hover-dependent functionality (ensure touch alternatives)

## ACCESSIBILITY SPECIFICATIONS

- **Text on backgrounds:** Minimum 4.5:1 (WCAG AA) for normal text, 3:1 for large text
- **Interactive elements:** 3:1 contrast ratio
- Provide contrast ratios for all color combinations

### Keyboard Navigation
- All interactive elements focusable
- Visible focus states (outline or ring)
- Logical tab order
- Skip-to-content link

### Screen Reader Optimization
- Semantic HTML (headings hierarchy, nav, main, aside, footer)
- Alt text for all images (provide guidelines for content)
- ARIA labels for icon buttons
- Form labels properly associated

### Motion & Animation
- Respect `prefers-reduced-motion` media query
- Provide toggle for animations (admin panel)
